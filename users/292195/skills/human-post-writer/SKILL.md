# Human Post Writer Skill

Write posts that sound like a real person wrote them — not a language model.

## Core principle

Every sentence must pass the "would a real person write this?" test. If it sounds like a summary, a report, or a template — rewrite it.

---

## FORBIDDEN patterns (never use these)

**Filler openers:**
- "В современном мире...", "В эпоху цифровизации...", "Ни для кого не секрет..."
- "Сегодня я хочу поговорить о...", "Давайте разберёмся..."
- "It's no secret that...", "In today's world..."

**Filler closers:**
- "Подводя итог...", "В заключение...", "Таким образом..."
- "Надеюсь, эта статья была полезна", "Буду рад вашим комментариям"

**Hollow affirmations:**
- "Безусловно", "Несомненно", "Важно отметить", "Стоит подчеркнуть"
- "Absolutely", "It's worth noting", "It's important to mention"

**AI structure tells:**
- Intro → 3 balanced points → conclusion
- Every paragraph same length
- Bullet lists where prose would be natural
- Hedging every opinion: "мне кажется", "возможно", "на мой взгляд" (once is fine, every sentence is not)

**AI vocabulary:**
- "комплексный подход", "синергия", "экосистема продукта"
- "leverage", "paradigm shift", "holistic approach"
- Passive voice as default

**Punctuation:**
- Em-dash (—) is forbidden everywhere. Replace with a period, comma, or restructure the sentence.

---

## REQUIRED techniques

**Sentence rhythm — mix short and long:**
- Drop a one-liner. Then let a longer thought run, clarify itself, maybe contradict the opener slightly — that's how thinking actually works.
- Never write five sentences of the same length in a row.

**Specificity over generality:**
- Bad: "Продукт показал хорошие результаты"
- Good: "За три недели — 40 продаж. Без рекламы."

**Direct opinions, no apology:**
- Bad: "На мой взгляд, возможно, этот подход мог бы быть более эффективным"
- Good: "Это не работает. Проверено."

**Self-interruptions and tangents (use sparingly):**
- "...хотя стоп, это отдельная история"
- "(потом расскажу почему это важно)"
- "ладно, не об этом"

**Rhetorical questions as transitions:**
- "Почему так? Потому что..."
- "И что с того?"

**Platform tone:**
| Platform | Tone |
|---|---|
| Telegram channel | Casual, direct, personal, short bursts ok |
| Instagram | Visual-first, emotional, punchy opener |
| LinkedIn / VC.ru | Professional but still human — story > advice |
| Twitter/X | Ultra-short, opinion, wit |

---

## Process

1. **Identify**: topic, platform, target reader, desired emotion (inspire / inform / entertain / sell)
2. **Draft**: write without AI templates — start mid-thought if needed
3. **Self-check**: scan for any FORBIDDEN patterns above → replace
4. **Rhythm check**: read aloud mentally — does it flow like speech?
5. **Output**: post only, no meta-commentary like "вот ваш пост:"

## Examples

### ❌ AI version
> В современном мире важно уметь правильно расставлять приоритеты. Многие люди сталкиваются с проблемой тайм-менеджмента. В данной статье мы рассмотрим ключевые подходы к эффективному планированию. Безусловно, каждый из них имеет свои преимущества.

### ✅ Human version
> Я убил три года на todo-листы, матрицы Эйзенхауэра и notion-доски.
>
> Работает одно: написать вечером три дела на завтра. На бумаге. Больше трёх — ложь себе.
>
> Всё остальное — продуктивность для галочки.

---

### ❌ AI version
> Наша команда провела масштабную работу по оптимизации процессов. Результаты превзошли ожидания.

### ✅ Human version
> Три месяца переписывали онбординг. Думали, займёт две недели.
>
> Конверсия выросла на 18%. Не потому что стало красивее — просто убрали один лишний экран.
