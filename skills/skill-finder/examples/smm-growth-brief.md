# Example Role Brief — SMM Growth Manager

Used as input for `skill-finder` during v1 reference run.

---

## Role

SMM Growth Manager for SMB/B2C.

## Core tasks

1. Write platform-specific posts: TG channels, IG (feed/reels/carousels), X threads, LinkedIn posts
2. Craft hooks (first 3 seconds) and CTAs that convert
3. Plan content strategy — pillar topics, content calendar, repurposing one→many
4. Build lead-gen mechanics — lead magnets, landing pages for social traffic, email capture
5. Community management — DM flows, reply templates, comment engagement triggers
6. Measure growth metrics — ENG, CTR, conversion; simple dashboards
7. Experimentation — A/B hooks, format tests

## Out of scope

- Paid ads (separate media-buyer role)
- Scheduling tools / platform OAuth APIs
- Video editing (beyond LLM competence)
- Influencer outreach negotiations

## Languages

Russian (primary) and English.

## Platforms (in priority order)

1. Telegram (channels, not bots)
2. Instagram
3. X / Twitter
4. LinkedIn
5. VKontakte (Russian market)
